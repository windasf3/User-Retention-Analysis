### User Retention Analysis in Retail Dataset

This project aims to analyze user retention in a retail dataset using cohort analysis. The dataset contains transactional data of customers over a period of time. The code provided in this repository includes preprocessing steps, handling missing values, and conducting cohort analysis to determine the customer retention rate.


#### Results

The results of the user retention analysis will be presented in the `results` directory. This includes visualizations such as cohort retention curves and metrics indicating the retention rate over time.
